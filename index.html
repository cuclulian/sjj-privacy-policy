《少女机甲舰》隐私政策

更新日期：2026年6月9日　生效日期：2026年6月9日

《少女机甲舰》（以下简称“本游戏”或“我们”）由菌菇游戏工作室运营。我们非常重视用户的个人信息保护与隐私安全。本隐私政策将向您说明我们如何收集、使用、共享、存储与保护您的个人信息，以及您享有的相关权利。

一、我们如何收集和使用个人信息

我们将遵循合法、正当、必要原则，仅在实现产品功能所必需的范围内处理您的个人信息。您在首次启动本游戏时，我们会向您展示隐私政策并征求您的明示同意；在您同意之前，本游戏不会初始化会收集设备标识、网络信息的第三方 SDK。

二、第三方 SDK 信息共享清单

为实现账号登录、云存档、广告展示与运营统计等能力，我们可能接入第三方 SDK。第三方 SDK 可能会收集您的相关信息，具体如下：

  • Tap SDK（TapTap SDK）
    第三方名称：易玩（上海）网络科技有限公司（TapTap）
    收集信息类型：Android ID
    处理目的：用于 TapTap 登录、云存档等相关服务的设备标识与安全风控、故障排查与服务稳定性保障
    处理方式：在您同意本隐私政策后初始化并通过网络传输至第三方服务器处理
    官方隐私政策链接：https://developer.taptap.cn/docs/sdk/start/agreement/
  • TapADN SDK（TapAd SDK）
    第三方名称：Dirichlet（TapTap 广告相关服务）
    收集信息类型：Android ID、IP 地址、传感器信息
    处理目的：用于广告展示与效果统计、反作弊与安全风控、问题排查与服务稳定性保障
    处理方式：在您同意本隐私政策后初始化并通过网络传输至第三方服务器处理
    官方隐私政策链接：https://ssp.dirichlet.cn/docs/agreement/

三、您的权利

您可通过卸载本游戏或在系统层面清理应用数据来撤回同意；撤回同意后，我们将停止处理基于同意的相关信息，但法律法规另有规定或为实现必要功能所必需的除外。
